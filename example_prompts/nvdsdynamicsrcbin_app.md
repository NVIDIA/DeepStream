Use DeepStream SDK pyservicemaker APIs to develop the python application that can do the following.
- create a nvdsdynamicsrcbin application. link the source with fakesink directly.
- add four video files to the source. When playing the second file, add the first file and remove the third file.
- count the number and fps of raw buffers on source's src pad.
- Send a stop signal to the source after 10 seconds.


**Important**
Save the generated code in nvdsdynamicsrcbin_app directory.
Also generate a README.md with setup instructions and how to run the application.
